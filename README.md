# Portofolio Fithri Nur Faizah

Website portofolio siswa RPL berbasis HTML, CSS, dan JavaScript murni.

## Menjalankan

Buka `index.html` di browser, atau gunakan ekstensi Live Server di VS Code. Tidak ada instalasi dependensi yang diperlukan.

## Personalisasi

- Ubah identitas, riwayat, dan tautan sosial di `index.html`.
- Isi Instagram, GitHub, LinkedIn, dan email pada objek `profile` paling atas di `js/script.js`. Setelah itu, ikon di bagian **Kontak** akan otomatis membuka data kamu.
- Isi atau ganti proyek di array `projects` dalam `js/script.js`. Setiap proyek dapat memiliki `title`, `image`, `desc`, `tech`, `status`, `features`, `challenge`, `solution`, `demo`, dan `github`.
- Form kontak mengirim ke email pada `profile.email` melalui FormSubmit. Setelah pengiriman pertama, buka email aktivasi dari FormSubmit dan klik tautan konfirmasi agar pesan berikutnya masuk ke Gmail.
- Ubah data proyek pada array `projects` di `js/script.js`.
- Warna tema tersentral di variabel `:root` dalam `css/style.css`.
